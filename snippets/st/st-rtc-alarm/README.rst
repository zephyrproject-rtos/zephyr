.. _st-rtc-alarm:

ST RTC ALARM Snippet (st-rtc-alarm)
#################################

Overview
********

This snippet enables driver support for RTC alarms.
It also enables driver support for RTC clock calibration.

Requirements
************

Support for:

- :kconfig:option:`CONFIG_RTC_ALARM`
- :kconfig:option:`CONFIG_RTC_CALIBRATION`
