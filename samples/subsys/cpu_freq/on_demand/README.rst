   Demonstrates On-demand CPU frequency scaling policy.

Overview
********

This sample demonstrates the CPU frequency subsystem' on-demand policy.
The sample will print debug information like CPU load and CPU Frequency Scaling
Subsystem logs to the console for visibility into the subsystem.

The example will iterate through some simulated CPU processing scenarios in the
main application.
