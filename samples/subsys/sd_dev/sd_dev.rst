.. zephyr:code-sample-category:: sd_dev
   :name: SD Device

These samples demonstrate the use of SD Device support in Zephyr RTOS.

The SD Device subsystem provides device-side implementation for SD/SDIO/MMC
protocols, allowing Zephyr-based devices to act as SD peripherals rather
than SD hosts.

All SD Device samples can be found under :zephyr_file:`samples/subsys/sd_dev`
in the Zephyr project tree.
