.. zephyr:board:: m5stack_atoms3

Overview
********

M5Stack AtomS3 is an ESP32-based development board from M5Stack.

Hardware
********

The board peripherals:

- 8MB of Flash
- LCD IPS TFT 0.85", 128x128 px screen (ST7789 compatible)
- 6-axis IMU MPU6886
- Infrared emitter
