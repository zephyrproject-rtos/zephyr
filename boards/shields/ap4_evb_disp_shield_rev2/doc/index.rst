Apollo4 Plus EVB Display Shield Rev2
###############

Overview
********

Support for Apollo4 Plus EVB expansion boards (shields)
controller.

Programming
***********

Set ``--shield ap4_evb_disp_shield_rev2`` when you invoke ``west build`` or ``cmake``
in your Zephyr application.
