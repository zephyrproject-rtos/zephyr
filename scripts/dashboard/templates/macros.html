{# copy_button(target_id) Renders a small copy-to-clipboard button that copies the text content of the element with id="{{ target_id }}" when clicked. The button shows a check icon briefly on success. Code in clipboard.js (loaded globally by base.html) enables all copy buttons via the data-copy-target attribute. Usage: {% from "macros.html" import copy_button %} some text{{ copy_button("myElement") }} #} {% macro copy_button(target_id) %}{% endmacro %}
